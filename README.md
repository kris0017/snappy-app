Make sure that you already have your environment set up (i.e Node.js and MongoDB is installed).

# Snappy Backend

Navigate to snappy-backend

Install modules
```
npm install
```
Start server
```
npm start
```

# Snappy Web

Navigate to snappy-web

Install modules
```
npm install
```
Start server
```
npm start
```
Check connection
```
curl http://localhost:3000
```
